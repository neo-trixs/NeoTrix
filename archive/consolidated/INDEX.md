## 查询速查

```
# 我的身份
cat SELF.md

# 今天的教训
rg "CRITICAL" TIMELINE.md

# 一个具体规则
rg "死代码复活" RULES.md

# 跨所有记忆查找一个主题
rg "编译" archive/consolidated/

# 查看自信度最高的知识
rg "1\.00" RULES.md | head -10

# 查看关联图的一个节点
rg "死代码" GRAPH.md

# 按日期精确回溯
rg "^2026-06-24 \| P5" TIMELINE.md

# 查看所有MEDIUM以下的知识 (需要小心使用)
rg "^[A-Z]" RULES.md | rg "0\.[5-7]\d" | head -10
```
