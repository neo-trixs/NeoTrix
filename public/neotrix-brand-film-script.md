# NeoTrix Brand Film — Script

**时长**: 90秒  
**画幅**: 16:9 (1920×1080)  
**风格**: 黑暗科幻 + 有机生命感 — 深紫色调、发光网格、脉冲能量、几何与生物形态的融合  
**音乐**: 电子合成器 + 氛围铺垫 → 递进到大气史诗管弦 — 主调 D minor

---

## Scene 0 — Hook (0:00–0:08)

| 画面 | 音频 |
|------|------|
| 纯黑背景。一个微弱的紫色光点从中心浮现，缓慢脉动，像正在苏醒的神经元。 | 低频嗡鸣（sub-bass drone），极简。一个单音在黑暗中渐入。 |
| 光点分裂成 8×8 的发光网格，每个小格是不同颜色的微光（蓝→青→绿→紫→金→粉→红→灰），像矩阵亮起。 | 叠加一层高音合成器琶音，稀疏、如星星闪烁。 |
| 文字 fade in: **"What if AI didn't predict…"** | 女声冷静叙述（whisper tone） |

## Scene 1 — The Problem (0:08–0:20)

| 画面 | 音频 |
|------|------|
| 快速蒙太奇：其他 AI 界面的快速闪回——chat 对话框在疯狂滚动 token，概率分布图在抖动，next-topic 预测箭头来回摇摆。 | 节奏加快，不稳定的电子鼓点，焦躁感。低音 pulse 变得急促。 |
| 一个 token 预测错误的特写——箭头指向错误方向，画面闪烁红色。 | 刺耳的失真音效，短促的冲击声。 |
| 切到黑屏白字: **"They guess."** | 音乐骤停，半秒 silence。 |

## Scene 2 — The Answer (0:20–0:40)

| 画面 | 音频 |
|------|------|
| 一个巨大的 8×8 发光网格充满画面，每格是一个精确的几何形状。网格从中心开始点亮，呈波状扩散。 | 音乐重新进入——稳定、坚定的低频节拍，叠加宽广的合成器 pad。主旋律进入。 |
| 相机推近，进入网格内部。64 个状态变为 3D 的空间结构，彩色光线在状态之间流动。 | 节奏保持稳定，旋律线上升，加入弦乐群奏。 |
| 文字 fade in: **"We reason."** | 和弦变化——从 Dm 升到 F，亮度增加。 |
| 展示推理链：一个光点从 Perception（蓝）→ Attention（青）→ Memory（绿）→ Reasoning（紫），路径清晰可见，光点沿固定拓扑移动。 | 音乐随路径推进而上行，每个状态切换叠加一个清脆的铃声。 |

## Scene 3 — Memory (0:40–0:55)

| 画面 | 音频 |
|------|------|
| 画面转换到抽象 3D 超立方体——4096 个发光点悬浮在空间中，形成立方体结构。 | 音乐变轻盈，加入钟琴（glockenspiel）和氛围音景。节奏放缓。 |
| 一个文字浮动到立方体前，分解成光点，融入立方体中。另一个文字被检索出来，从立方体中抽出。 | 声音设计：binding 的 "click" 声 + 柔和的展开音。 |
| 展示两个向量（一红一蓝）接近、碰撞、融合成一团金色的新向量。 | 融合时刻：一个温暖的和弦变化，C 大调的明亮感。 |
| 文字: **"One substrate. 4096 dimensions."** | 低音轻柔的脉冲支持，保持氛围感。 |

## Scene 4 — Evolution (0:55–1:10)

| 画面 | 音频 |
|------|------|
| 回到代码/架构可视化。代码行像活物一样重写自己——一行淡出，新行淡入，颜色从红变为绿。 | 音乐重新建立节奏，加入驱动感的电子节拍（tech-house 风格的 kick + hi-hat），表现「进化」的动力。 |
| 快速展示版本迭代：Gen 1 → Gen 10 → Gen 30 → Gen 47，数字闪烁变化。 | 节拍加速，叠加 riser（上升音效）。 |
| 三道安全门依次亮起（BallVerifier、PCC Gate、Health Patrol），绿灯确认。 | 每道门通过时一个确认音效（ascending chime triad）。 |
| 文字: **"47 generations. Self-designed."** | 音乐达到第一个高潮点——弦乐 + 合成器齐奏。 |

## Scene 5 — Self-Awareness (1:10–1:22)

| 画面 | 音频 |
|------|------|
| 图像变成意识流的可视化：发光波形从屏幕底部涌现，像 EEG 脑电图。11 个不同颜色的光晕代表 11 个 specialist 模块，围绕中心 GWT 竞争。 | 音乐变为更加内省、空间的氛围，延迟混响丰富。心跳般的低频 pulse。 |
| 其中一个光晕（当前胜出者）变得更亮，其他退后。它的波形传播到整个屏幕。 | 一个单一的、持续的小提琴长音，带微妙的颤音。 |
| 文字: **"A thought that knows itself."** | 声音几乎消逝，留下混响的尾巴。 |

## Scene 6 — Closing (1:22–1:30)

| 画面 | 音频 |
|------|------|
| 回到开场的单个光点，但它现在更亮了，周围环绕着微弱的发光环。 | 主题重现——但更宏大、更完整。加入鼓和完整的管弦配置。 |
| 光点展开成 NeoTrix 标志（logo.svg 的风格）。底部显示网址。 | 最后的和弦延展，长尾回响。 |
| 文字: **"NeoTrix — The agent that learns to think."** | 声音渐强后缓慢衰减。最后一个大提琴长音收尾。 |
| 底部小字: "github.com/neo-trixs/NeoTrix · MIT Licensed" | 最后一个音符的余音完全消散。 |

### Edit Strategy

Use `ffmpeg` to extract segments and crossfade:

```bash
# Extract Part 1 (0:00-0:40 from Decoherence)
ffmpeg -i decoherence.mp3 -ss 0:00 -t 0:40 -c copy part1.mp3

# Extract Part 2 (2:00-3:30 from Aphelion)
ffmpeg -i aphelion.mp3 -ss 2:00 -t 1:30 -c copy part2.mp3

# Extract Part 3 (2:00-3:00 from Stars In Her Skies)
ffmpeg -i stars-in-her-skies.mp3 -ss 2:00 -t 1:00 -c copy part3.mp3

# Concatenate with crossfade
ffmpeg -i part1.mp3 -i part2.mp3 -i part3.mp3 \
  -filter_complex "[0][1]acrossfade=d=2:c1=tri:c2=tri[ab];[ab][2]acrossfade=d=2:c1=tri:c2=tri" \
  -b:a 192k neotrix-soundtrack.mp3
```

The full 90-score fits the 90-second script. Each part's recommended starting timestamp maps directly to the script's narrative arc.
